## 7. Performance SEO

- **Images** : Format WebP + fallback PNG, lazy loading (`loading="lazy"`)
- **Fonts** : Preload Inter et JetBrains Mono
- **CSS** : TailwindCSS purgé en production
- **JavaScript** : Code splitting par section
- **Core Web Vitals** : LCP < 2.5s, FID < 100ms, CLS < 0.1
- **HTTPS** : Obligatoire
- **Compression** : Gzip/Brotli
